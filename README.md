# Twig PHP-ref extension

A extension for the Twig template engine. Adds php-ref functions/filters to Twig.  
A better alternative for the standard debug filter.

## Usage

```twig
<h1>User debug:</h1>
{{ user|r }}
```

see the examples folder for a full example.
